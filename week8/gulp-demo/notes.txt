Things Gulp Does
	- Compile Pre-processors (Stylus, Coffeescript, etc)
	- Minification -> can minify both css and js
	- Concatenation -> 12 js files : 1 js file
	- Run tests
	- Watch files for changes

Install Gulp
	make sure your package.json.name isn't 'gulp'
	global - <sudo> npm install -g gulp
	local  - npm install gulp --save-dev

To create a Gulpfile
	- Gulpfile.js
	- gulpfile.js
